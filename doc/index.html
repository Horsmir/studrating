Введение

Программа StudRating предназначена для учёта рейтинга студентов по различным дисциплинам, а также вычисления и вывода итоговых результатов.

Основные функции:

  • ведение журнала учёта рейтинга по различным критериям;
  • настройка критериев, по которым ведется учёт;
  • ввозможность вести учет по разным дисциплинам в нескольких группах;
  • автоматический расчёт суммарного рейтинга по всем критериям.
